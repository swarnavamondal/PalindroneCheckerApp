PalindromeChecker App
Overview

The PalindromeChecker App is a console-based Java application that checks whether a given string is a palindrome. The program evaluates the string under multiple conditions to demonstrate fundamental programming concepts and string manipulation techniques.

Objective

The objective of this project is to:

Validate whether a given string is a palindrome.

Practice Java programming fundamentals.

Apply string manipulation and basic data structure concepts.

Implement multiple validation conditions for palindrome checking.

Features

The application performs palindrome validation in three different ways:

Basic Palindrome Check

Direct comparison of the string with its reverse.

Case-Insensitive Check

Ignores differences between uppercase and lowercase characters.

Ignore Spaces and Special Characters

Removes punctuation, spaces, and special characters before checking.

Technologies Used

Java

Java Standard Library

Scanner Class for user input

StringBuilder for string reversal
